# 💊 Smart Medicine Reminder System

A **Design Thinking based healthcare web application** designed to help elderly people manage their medicines, doctor appointments, and emergency contacts efficiently.

---

## 📌 Project Overview

Many elderly people forget to take medicines on time due to busy schedules or memory-related issues. Missing medications can lead to serious health problems.

This project provides a **simple and user-friendly dashboard** that allows users to track medicines, manage appointments, and store emergency contacts.

The system was developed using the **Design Thinking process**:

Empathize → Define → Ideate → Prototype → Test

---

## 🚀 Features

- 🔐 Secure **Sign In / Sign Out authentication**
- 💊 **Medicine reminder dashboard**
- ✔️ Mark medicines as **Taken or Skipped**
- 📅 **Doctor appointment scheduling**
- 🚨 **Emergency contact management**
- 📊 **Medication history tracking**
- 👴 Elderly-friendly interface
